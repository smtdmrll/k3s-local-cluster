🚀 Deploy K3s to Remote Server

Install: SSH into the server → Clone repo → Run k3s-project.sh install → K3s + Helm + Sealed Secrets + PostgreSQL + Redis.
Delete: SSH into the server → Run k3s-project.sh delete → Remove entire cluster and all workloads.
Note: The server must have Ubuntu and the SSH user must be devops with sudo access. Login App is not deployed — use install-withapp manually for full stack.
